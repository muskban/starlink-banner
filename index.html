Test page for starlink-banner

Purpose

This page demonstrates the usage of the javascript library startlink-banner.

This library detects clients using Starlink to access the web, and shows a banner warning them that this company belongs to Elon Musk, who is openly a neo-Nazi.

Preview

The banner looks like this :

Setup

Just include the following line in your html/head:

<script type="module" src="https://cdn.jsdelivr.net/npm/starlink-banner/dist/bundle.min.js"></script>

Test

Add "?starlink=1" on any URL to see the banner.
